сайт для торговли между физическими лицами.

Интерфейс: 
Интерфейс по типу инстаграмма, только вместо новостей 
человек выкладывает продукт для продажи.
Есть окно аккаунта, окно ленты с товарами.
Также присутствует окно авторизации, окно заполнения
формы продукта для продажи, окно покпки и окно 
пополнения баланса.

Особенности работы:
1.Человек на своём аккаунте может прикрепить фото профиля.
2.Кошелёк, через который проходит оплата товара.
